For the user:

This is an online website designed to automate the process of paying toll at a toll plaza using online mediums. Firstly, the user needs to create an account which will be linked to the user's eWallet.


Note: The bank account credentials will be required in the signup process of the Toll Plaza.


After the account is created, the user needs to move ahead to create an account in the toll plaza's main website, by clicking on the signup button present in the navbar of the website. The signup form will ask the user to enter the necessary credentials i.e.
  1. Username
  2. eWallet number
  3. Email id
  4. Car number
  5. Password
After completing the signup procedure, the user can come back to the homepage and signin to his/her account. , where the user will be asked to enter username, password and the car number. The user will the be directed to a page which displays the toll amount and a 'payment button' But, the user first needs to transfer some amount of money to the eWallet from his/her bank account(Initially the eWallet balance is set to 0 Rupees) Click on the credit money to eWallet button. The user will be asked to enter the New Horizon Bank account number and pin provided by the bank. Then, the user can enter the amount that he/she wants to transfer from account to eWallet and then click on submit. The money transfer will take place, if the money available in the bank account is sufficient for the transfer. Then the user can click on the 'Pay now' button to make a transaction. If the eWallet balance is sufficient, then 150 Rupees will be deducted from the eWallet and a message showing successful transaction will show up. The user can now drive out of the toll plaza. The user can also see this and the previous transaction by clicking on the transaction history button after sigining into his/her account.

For the administrator:

The adminstrator can sign into his/her account by clicking on the administrator button on the main webpage. Then, administrator needs to enter administrator name and the password After that, the administrator can view the transaction details of every transaction that happened on the toll plaza in the from of a table, and also the total revenue collected so far at the toll plaza, just below the table.
